##  About the Project
This project is based on the study of Python as an open-source software. It explores how Python was created, the type of license it uses, and how it fits into the open-source ecosystem.

The project also includes a practical section where shell scripts are developed to perform basic system-level tasks. These scripts demonstrate how open-source tools can be used effectively through command-line operations.

The goal of this project is to understand both the theoretical concepts and practical applications of open-source software.

---

## Shell Scripts Summary

**Script 1: System Report Generator**  
Displays general system details such as user name, kernel version, and date.

**Script 2: Software Checker**  
Verifies whether Python is installed and shows its version.

**Script 3: Directory Scanner**  
Checks selected directories and reports their size and permission details.

**Script 4: Log Analyzer**  
Scans a log file, counts keyword occurrences, and displays matching entries.

**Script 5: Manifest Creator**  
Collects user input and generates a custom open-source statement saved in a file.
